# AR & VR Based Object Information Display Application

This project explores the integration of Augmented Reality (AR), Virtual Reality (VR), and Artificial Intelligence (AI) to create immersive systems for object identification and interactive learning.

## 🎯 Project Objective
The primary goal is to make camera interfaces smarter and more interactive. By using AR, the application overlays digital labels directly onto real-world objects, providing users with instant access to information without the need for manual searches.

## 🛠️ System Workflow
The current prototype follows a structured AR process:
* Environment Scanning: The application uses the mobile camera to scan the real-world environment.
* Interaction: The user taps on a detected surface or specific object position.
* AR Anchoring: The system creates a fixed digital anchor at the selected position.
* Information Overlay: A digital label appears above the object showing its name, type, or extra details.

## 📊 Research Findings
The project conducted a study with engineering students to evaluate how these technologies improve the understanding of abstract Python data types (Lists, Tuples, Sets, Dictionaries).

* Learning Hierarchy: Research demonstrated that AI-enhanced VR produced the highest learning impact, followed by standard VR and AR.
* Student Performance: Students using AI-enhanced VR achieved significantly higher scores in comprehension assessments compared to traditional paper-based methods.
* Engagement: EEG analysis confirmed that higher levels of immersion and attention directly correlated with better academic performance.
* AI Integration: The use of an AI virtual assistant helped reduce the difficulty of abstract concepts through real-time feedback and personalized guidance.

## 🚀 Future Use Cases
The technology is designed to be expanded into various professional fields:
* Automotive: Identifying car types, conditions, and estimated prices.
* Retail/Shopping: Instant display of product names and details.
* Education: Helping students identify objects and learn through visual interaction.
* Museums: Providing historical details and context for exhibits.
